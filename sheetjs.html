A basic JAVASCRIPT cheatsheet

Intro Access DOM Modify DOM

Access elements from DOM

From Document

Find an element by ID

document.getElementById("myid");

Find all elements by class

document.getElementByClassName("myclass");

Find all elements by HTML Tag

document.getElementsByTagName("myhtmltag");

From element

Using same methods

let myElement = document.getElementByClassName("myclass");
myElement.getElementsByTagName("myhtmltag");

Access children elements

let data = myElement.children;

Access parent elements

let data = myElement.parentElement;

Access next element

let data = myElement.nextElementSibling;

Access previous element

let data = myElement.previousElementSibling;

Query selector

The Document method querySelector returns the first Element within the document that matches the specified selector, or group of selectors. 

document.querySelector("#myid");
document.querySelector(".myclass");
document.querySelector("mytag");
document.querySelector("#myid mytag.myclass > mytag");

For all elements

document.querySelectorAll("#myid");

Modify elements from DOM

Modify content

Element content modification

let e = document.getElementById('myid');
e.innerHTML = "<ul><li>data 1</li><li>data 2</li></ul>";
e.textContent = "just some text but not html";

Modify class

Add a class

e.classList.add("myClass");

Remove a class

e.classList.remove("myClass");

Replace a class

e.classList.replace("myClass");

Check if contains a class

e.classList.contains("myClass");

Modify style

For an element you can access to the object style and modify all properties.

e.style.color = "#fff";
e.style.backgroundColor = "#000";
e.style.fontWeight = "bold";

Element Manipulation

Create an element add it to a document element with appendChild

const newElt = document.createElement("div");

Add element newElt to element elt

let elt = document.getElementById("myid");
elt.appendChild(newElt);

Remove element newElt from element elt

elt.removeChild(newElt);

Replace element oldElt with newElt to element elt

elt.replaceChild(newElt, oldElt);
